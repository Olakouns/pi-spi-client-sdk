# Gouvernance du SDK PI-SPI

## Modèle de projet
Ce projet est un projet Open Source dirigé par son créateur (**Benevolent Dictator For Life**).

## Rôles
- **Lead Maintainer** : @Olakouns (Razacki KOUNASSO). A un contrôle total sur le dépôt, les releases et la direction technique.
- **Contributors** : Toute personne ayant au moins une Pull Request acceptée.

## Prise de décision
Les décisions techniques sont prises via les discussions dans les **Issues** et les **Pull Requests**. En cas de désaccord, le Lead Maintainer a le dernier mot.

## Transfert de propriété
Si le Lead Maintainer ne peut plus assurer la maintenance pendant plus de 6 mois, la propriété peut être transférée à un contributeur actif après discussion avec la communauté.
